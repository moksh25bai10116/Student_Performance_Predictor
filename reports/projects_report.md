# Student Performance Predictor

## Introduction
Predicting student performance helps identify at-risk students early and enables timely intervention.

## Problem Statement
To predict final student scores using machine learning and classify risk levels.

## Objectives
- Build regression model
- Classify risk levels
- Compare models

## Dataset
A synthetic dataset was generated using statistical distributions to simulate real-world student data.

## Methodology
- Data preprocessing
- Feature scaling
- Encoding categorical data
- Model training (Linear Regression, Decision Tree)
- Evaluation using MAE, RMSE, R²

## Results
Linear Regression performed better due to stable relationships between variables.

## Challenges
- Data realism
- Feature selection
- Avoiding overfitting

## Learnings
- Importance of preprocessing
- Model comparison insights
- Real-world ML workflow

## Future Improvements
- Use real datasets
- Add web app
- Use advanced models
