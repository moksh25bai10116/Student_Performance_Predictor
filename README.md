# Student Performance Predictor

## Overview
This project predicts a student's final academic performance using machine learning and classifies them into risk categories.

## Features
- Predict final score
- Risk classification (Low, Medium, High)
- Data preprocessing pipeline
- Model comparison

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Matplotlib

```bash
git clone <your-repo-link>
